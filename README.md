# 📊 analyTiX

> A real-time analytics dashboard built with Vue.js 3 — a learning project for practicing modern frontend development.

## 🎯 About

This is an **activity/project** to demonstrate real-time data visualization and Vue.js 3 fundamentals. The dashboard updates automatically every 3 seconds, simulating live data.

## ✨ Features

- Live KPI tracking
- Dynamic bar chart
- Activity feed
- Responsive design

## 🛠️ Tech Stack

- Vue.js 3
- CSS3
- Font Awesome
- JavaScript
